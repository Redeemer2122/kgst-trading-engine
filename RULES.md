# 📜 RULES.md — Глобальные правила KGST_Project

## 🛠 Технический стек

- **Язык:** Python 3.13.5
- **Библиотеки:** aiogram (Telegram), ccxt (Exchange), python-dotenv (Secrets).
- **Окружение:** Virtualenv (venv), PowerShell 7.

## 🏗 Архитектура и стиль

- **Модульность:** Код разделен на monitor.py (запуск), engine.py (логика), database.py (БД), config.py (настройки).
- **Кодировка:** Строго UTF-8 для всех файлов и логов.
- **Безопасность:** Ключи и токены берутся ТОЛЬКО из .env.
- **Логирование:** Все важные события и ошибки записываются в bot.log.

## 🤖 Правила для Исполнителя (Builder)

1. ПЕРЕД началом работы всегда читай этот файл и текущий stepX.md.
2. ПЕРЕД изменением кода делай проверку на ошибки.
3. ПОСЛЕ завершения задачи проверяй запуск бота командой python monitor.py.
4. Никогда не удаляй существующие комментарии, если это не прописано в ТЗ.
